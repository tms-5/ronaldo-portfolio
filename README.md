# Ronaldo Neto | Portfólio

Site portfólio desenvolvido para apresentar a atuação profissional de Ronaldo Neto na área de nutrição, com foco em ciência, estratégia, saúde metabólica e acompanhamento individualizado.

A proposta visual combina uma estética limpa, leve e institucional, usando tons naturais, tipografia elegante, animações sutis e seções pensadas para guiar a navegação em uma experiência de página única.

## Tecnologias

- Vue.js
- Vite
- Tailwind CSS
- Lucide Icons

## Sobre o projeto

O site é uma SPA, ou seja, todas as seções ficam em uma única página com navegação suave. Ele apresenta informações sobre formação, abordagem profissional, foco de atuação, conteúdos e contato.

As animações foram pensadas para deixar a navegação mais fluida, com elementos surgindo delicadamente conforme o usuário percorre a página.
